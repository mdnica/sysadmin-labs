# Networking Basics – SysAdmin View

## IP Address

A unique identifier for a machine on a network.

Example:
192.168.1.10

## Subnet

Defines which machines are local.

Example:
255.255.255.0

## Gateway

The router that connects us to the outside world.

Example:
192.168.1.1

## DNS

Translates names to IP addresses.

Example:
google.com → 142.250.74.14

## DHCP

Automatically gives IP addresses to machines.
